## Lab Rules

1. Use one project scenario across all labs.
2. Keep facts, assumptions, and conclusions separate.
3. Do not jump to countermeasures before checking root causes.
4. Save every artifact for the final A3 presentation.
5. Keep calculations visible for trainer review.
