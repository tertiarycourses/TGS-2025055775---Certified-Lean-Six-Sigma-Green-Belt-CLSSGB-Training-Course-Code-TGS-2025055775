# Tools and Templates Reference

## Free Tools

| Tool | Use in Course |
| --- | --- |
| Microsoft Excel | Data tables, formulas, run charts, Pareto charts, histograms. |
| LibreOffice Calc | Spreadsheet alternative for calculations and charts. |
| Google Sheets | Shared data collection and simple analysis. |
| diagrams.net | SIPOC, process maps, fishbone diagrams, and A3 visuals. |
| Whiteboard or sticky notes | Kaizen facilitation and brainstorming. |
| Paper templates | Project selection, A3, FMEA, control plan, and standard work. |

## Core Templates

| Template | Key Fields |
| --- | --- |
| Project Selection | Problem, customer impact, COPQ, feasibility, data availability, scope |
| SIPOC | Suppliers, inputs, process, outputs, customers |
| Process Map | Step, owner, handoff, wait, rework, value classification |
| Stakeholder Analysis | Stakeholder, interest, influence, concern, engagement action |
| A3 Report | Background, current condition, goal, analysis, countermeasures, plan, results |
| Data Collection Plan | KPI, definition, source, sample size, frequency, owner |
| FMEA | Failure mode, effect, cause, controls, severity, occurrence, detection, RPN |
| Control Plan | Metric, target, method, frequency, owner, response plan |

## Common Formulas

```text
Yield = Good Units / Total Units
DPU = Defects / Units
DPMO = (Defects / (Units * Opportunities)) * 1,000,000
RPN = Severity * Occurrence * Detection
Range = Maximum - Minimum
```

## Chart Selection Guide

| Need | Suggested Chart |
| --- | --- |
| Show performance over time | Run chart |
| Show largest defect categories | Pareto chart |
| Show distribution of continuous data | Histogram |
| Compare before and after | Bar chart or line chart |
| Show process flow | Flowchart or swimlane map |
